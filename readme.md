### Features 

A aplicação explora diversos recursos, como autenticação, permitir ações apenas ao usuário autenticado, rotas dinâmicas, segurança nas rotas, paginação, requisiçoes assíncronas e muito mais.

* PHP
  - PDO (SQL)
  - OOP
  - SPL - Autoload
* Javascript
  - Fetch API (Async)
  - Nullish Operator
  - Dropdown
  - Responsive Menu
* HTML/CSS
